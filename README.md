# SamayTracker
An AI-powered, proactive productivity companion that transforms passive reminders into actionable execution plans to prevent missed deadlines.

## Features

* **Proactive Task Decomposition:** Automatically breaks down complex goals into micro-tasks.
* **Adaptive Scheduling:** Dynamically updates your daily timeline when plans change.
* **Contextual Triggers:** Attaches relevant links, documents, or tools directly to task alerts.
* **Risk Assessment:** Flags high-risk deadlines based on your current workload.

## Tech Stack

🛠️ Tech Stack
Frontend: React (Vite)

Backend: Node.js, Express

AI Core: Google Gemini API via Google AI Studio

🚀 Run Locally
>Prerequisites
Node.js

>Setup Steps
1. Install dependencies:

npm install

2. Set the GEMINI_API_KEY in .env.local to your Gemini API key:

GEMINI_API_KEY=your_actual_api_key_here

3. Run the app:

npm run dev
